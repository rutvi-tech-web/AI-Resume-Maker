# AI Resume Maker Pro

AI-powered Resume Builder built using React, Node.js, Express, and Gemini AI.

## Features

* AI Resume Summary Generation
* AI Skill Recommendations
* ATS Score Checker
* ATS Suggestions
* PDF Resume Download
* Dark Mode
* Multiple Resume Templates
* Profile Photo Upload
* Local Storage Support

## Tech Stack

### Frontend

* React
* Vite
* CSS
* jsPDF

### Backend

* Node.js
* Express.js
* Gemini AI API
